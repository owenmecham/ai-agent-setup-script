Welcome to Murph

Your personal AI agent framework

This wizard will install and configure Murph on your Mac. It will set up all required dependencies, build the project, and configure automatic startup.

Prerequisites

Before we begin, please confirm the following:

Full Disk Access

Murph needs Full Disk Access to read the iMessage database. Because the agent runs via LaunchAgent, the node binary itself needs FDA — not your terminal app.

  1. Click Open Settings below
  2. Click the + button and add the node binary:
    /usr/local/bin/node
  3. Toggle the switch ON
  4. Click Verify to confirm access

Accessibility

Murph needs Accessibility access to send iMessages via AppleScript.

  1. Click Open Settings below
  2. Click the + button and add your terminal app (e.g. Terminal.app)
  3. Toggle the switch ON
  4. Click Verify to confirm access
